Automatically upload data

English中文(简体)

Click on the plugin icon and see the configuration panel.Enter your receiving api address. Enter the strictly formatted json data. Send a data collection request. It's that simple
点击插件图标，可以看到操作面板，输入你需要接收数据的api地址，输入需要发送的数据结构，只支持JSON数据，并且只支持POST请求，点击采集按钮。数据采集就是如此简单！
The api that sends the request must use the full path, based on the HTTP or HTTPS protocol, otherwise the data cannot be sent.
发送请求的api必须使用全路径，基于HTTP或者HTTPS协议，否则无法发送数据。
The data structure that needs to be sent must be in JSON format. If there are additional constants, use the key-value pairs of the string. If you need to collect the data list from the page as an object, the key name of the data object, and the css of the collection object. The selector acts as a key to form the acquisition object. Currently only supports 2D data structures, and will increase multidimensional data in the future.
需要发送的数据结构必须是JSON格式的，如果有附加常量，就使用字符串的键值对，如果需要从页面采集数据列表以对象的方式进行构建，数据对象的键名，及采集对象的css选择器作为键值，组成采集对象。目前只支持二维数据结构，以后会增加多维数据。
Click the capture button to complete the acquisition. All the data is only the original data of the page, and the logic that needs to be processed twice needs to be implemented on the business server. Very simple
点击采集按钮，完成采集。所有的数据只是页面的原生数据，需要进行二次加工的逻辑需要在业务服务端实现。很简单吧？
